# AuthKit

A simple iOS authentication app built with UIKit.

## Features

- Sign In
- Sign Up
- Keyboard handling
- Input validation

## Requirements

- iOS 15.0+
- Xcode 15+

## Building

Open `AuthKit.xcodeproj` in Xcode and run on a simulator or device.
